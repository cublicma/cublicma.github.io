# Repository Instructions

This repository hosts a GitHub Pages site and an automated paper tracker.

For the paper tracker:
- Keep user-facing tracker content in Chinese unless configuration says otherwise.
- Read `paper-tracker/config.json` before changing tracker outputs.
- Keep tracker JSON files valid and pretty-printed.
- Prefer original research sources over reposts.
- Be strict about relevance and merge duplicates for the same work.

Do not commit secrets, local Codex state, or API keys.
